## 录制建议

- **分辨率**：1280×720 或 1366×768，避免超过 GitHub 单文件 10MB
- **帧率**：10-15 fps（gif 体积线性增长，不必追求 30fps）
- **分镜**：每个 Tab 演示 ≤ 12 秒，焦点放在「输入问题 → 答案出现」的过程
- **字体**：浏览器缩放到 110-125%，让录制画面里的文字在 GitHub README 缩略图里也清晰
